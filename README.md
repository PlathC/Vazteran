# Vazteran

Vulkan based rendering engine for learning purposes. It main goal is to make available a simpler way 
to render my small projects related to computer graphics. 

Here is the roadmap :
| Feature                                                         | Progress  |
|-----------------------------------------------------------------|-----------|
| Render Graph                                                    | WIP       |
| Ui Library                                                      | WIP       |
| Geometry helping library                                        | To Do     |
| ECS                                                             | To Do     |
